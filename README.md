# Student Academic and Behavior Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard designed to analyze student academic performance, attendance, and behavior patterns. It transforms raw educational data into meaningful insights that help educators identify strengths, weaknesses, and areas requiring intervention.

The dashboard focuses on performance tracking, behavioral analysis, and data-driven decision-making.

---

## Project Objective

The objective of this project is to:

- Analyze student academic performance across subjects  
- Evaluate attendance and behavioral patterns  
- Identify high-performing and at-risk students  
- Support data-driven academic decision-making  
- Visualize insights through interactive dashboards  

---

## Dashboard Components

### Overall Dashboard

- Total Students  
- Average Score  
- Attendance Percentage  
- Behavior Count  

### Academic Analysis

- Average Score by Subject  
- Percentage Score by Subject  
- Performance trends across terms  

### Behavior Analysis

- Behavior count by type  
- Distribution of student behavior patterns  

### Student-Level Dashboard

- Individual student performance  
- Attendance percentage  
- Performance category (High / Medium / Low)  
- Student ranking  
- Total score  

---

## Filters and Interactivity

- Subject selection  
- Term selection (Term 1, Term 2, Term 3)  
- Class filter  
- Section filter  
- Student name filter  

---

## Key Insights

### Academic Insights

- Performance varies significantly across subjects  
- Certain subjects consistently show higher average scores  
- Low-performing subjects require targeted improvement  

---

### Attendance Insights

- Higher attendance is strongly linked to better academic performance  
- Students with low attendance tend to have lower scores  

---

### Behavior Insights

- Most students show positive behavior patterns  
- Behavioral incidents can indicate potential academic issues  

---

### Student Performance Insights

- High-performing students can be identified for advanced programs  
- Medium-performing students benefit from structured support  
- Low-performing students require focused intervention  

---

## Decision-Making Insights

- Helps educators identify at-risk students early  
- Enables targeted academic support strategies  
- Supports efficient resource allocation  
- Improves overall academic outcomes
